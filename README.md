✨ Sistema de Gestión de Biblioteca ✨
📖 Descripción

Este proyecto consiste en una aplicación de escritorio desarrollada en C# para administrar una biblioteca de manera sencilla y organizada.

El sistema permite registrar libros, gestionar préstamos y devoluciones, buscar información y controlar el estado del inventario. Además, valida los datos ingresados y muestra mensajes de alerta para guiar al usuario durante las operaciones principales.

La información se almacena en archivos de texto y se maneja mediante una interfaz amigable en Windows Forms.

🛠️ Tecnologías Utilizadas

Lenguaje: C#

Framework: .NET (Windows Forms)

IDE: Visual Studio

Tipo de aplicación: Escritorio
